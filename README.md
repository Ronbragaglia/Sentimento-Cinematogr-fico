# Análise de Sentimento Cinematográfico

![Python](https://img.shields.io/badge/Python-3.x-blue?logo=python&logoColor=white) ![Topic](https://img.shields.io/badge/Topic-nlp-green) ![License](https://img.shields.io/badge/License-MIT-yellow)

## Descrição

Sistema de análise de sentimento de críticas de filmes, classificando reviews como positivas ou negativas usando VADER e modelos de ML treinados em corpus de críticas.

## Funcionalidades

- Classificação de críticas como positivas ou negativas
- Uso do VADER para análise de sentimento baseada em léxico
- Modelo treinado em corpus de reviews de filmes
- Pré-processamento NLP completo
- Métricas de avaliação e matriz de confusão

## Stack Tecnológico

| Tecnologia | Descrição |
|---|---|
| Python | Linguagem principal |
| NLTK / VADER | Análise de sentimento |
| Scikit-learn | Modelos de classificação |
| Pandas | Manipulação de dados |

## Como Usar

1. Instale as dependências: `pip install nltk scikit-learn pandas`
2. Execute: `python sentimento_filmes.py`
3. Insira uma crítica para obter a classificação de sentimento
